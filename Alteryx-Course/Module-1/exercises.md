# Answers:
  > 1.Definition

Alteryx is a low-code/no-code analytics automation platform used to:

- Prepare data
- Clean data
- Transform data
- Analyze data
- Automate repetitive processes
- Generate reports

Instead of writing hundreds of lines of code, users drag and drop tools onto a workflow canvas.

Example

Without Alteryx:
```
Excel File
↓
Clean Data Manually
↓
Copy Data
↓
Merge Files
↓
Create Pivot Tables
↓
Create Reports
```
With Alteryx:
```
Excel File
↓
Workflow
↓
Clean
↓
Transform
↓
Analyze
↓
Output
```
One-click execution.
