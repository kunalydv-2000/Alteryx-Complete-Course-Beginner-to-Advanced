# Module 2: Installation & Environment Setup

## 📖 Module Overview

Before building workflows and performing analytics, it is essential to properly install and configure Alteryx Designer. A well-configured environment ensures smooth workflow development, efficient data processing, and optimal performance.

In this module, you will learn how to install Alteryx Designer, understand system requirements, activate the software, navigate the user interface, and configure your development environment. You will also create your first workflow and become familiar with the core areas of the Alteryx Designer workspace.

This module serves as the bridge between theoretical concepts and practical workflow development.

---

## 🎯 Learning Objectives

By the end of this module, you will be able to:

- Understand Alteryx system requirements
- Install Alteryx Designer successfully
- Activate and configure Alteryx Designer
- Navigate the Alteryx Designer interface
- Understand key workspace components
- Configure workflow settings
- Manage file connections and dependencies
- Create and run your first workflow
